1. Begynn med å installer Python bibliotekene med PIP: 

"pip install openai elevenlabs numpy sounddevice soundfile pynput requests"


2. Sørg for at Main program og APIkey.py er i samme mappe.


3. Dersom ikkje programmen vil kjøre prøv å åpne det som administrator via CMD:
    "Win + r" 
    Søk "CMD" trykk "enter"
    Skriv "Python C:\filepath\filename.py"
    "Enter" får å kjøre programmet.
    "Ctrl + c" får å avslutte programmet.
